## 一、返回结果封装

　　按照《阿里巴巴Java开发手册》的要求，服务端发生错误时，返回给前端的响应信息必须包含 **HTTP 状态码、errorCode、
errorMessage、用户提示信息**四个部分。我们将这个要求简化为包含**HTTP 状态码，errorMessage**两个部分，当然还要包含最重要的**消息体**。

　　1、定义状态码枚举类

```java
package com.zyg.result;

/**
 * @Author: zyg
 * @Date: 2023/4/26 16:45
 * @Version: v1.0
 * @Description: HTTP状态码枚举类
 */
public enum ResultCode {
    SUCCESS(200,"操作成功"),
    FAILED(500,"操作失败"),
    VALIDATE_FAILED(404,"参数检验失败"),
    UNAUTHORIZED(401,"暂未登录或Token已经过期"),
    FORBIDDEN(403,"没有相关权限");

    /**
     * HTTP状态码
     */
    private long code;

    /**
     * HTTP状态码的解释短语
     */
    private String message;

    ResultCode(long code, String message) {
        this.code = code;
        this.message = message;
    }

    public long getCode() {
        return code;
    }

    public String getMessage() {
        return message;
    }
}
```

　　2、定义通用返回结果类

```java
package com.zyg.result;

/**
 * @Author: zyg
 * @Date: 2023/4/26 16:50
 * @Version: v1.0
 * @Description: 通用返回结果
 */
public class CommonResult<T> {
    
    /**
     * HTTP状态码
     */
    private long code;

    /**
     * 提示信息
     */
    private String message;

    /**
     * 返回的数据
     */
    private T data;

    public CommonResult(long code, String message, T data) {
        this.code = code;
        this.message = message;
        this.data = data;
    }

    /**
     * 成功返回结果
     * @param data 返回的数据
     */
    public static <T> CommonResult<T> success(T data) {
        return new CommonResult<>(ResultCode.SUCCESS.getCode(), ResultCode.SUCCESS.getMessage(), data);
    }

    /**
     * 成功返回结果，可以自定义提示信息
     * @param data 返回的数据
     * @param message 提示信息
     */
    public static <T> CommonResult<T> success(T data, String message) {
        return new CommonResult<>(ResultCode.SUCCESS.getCode(), message, data);
    }

    /**
     * 失败返回结果
     */
    public static <T> CommonResult<T> failed() {
        return new CommonResult<>(ResultCode.FAILED.getCode(), ResultCode.FAILED.getMessage(), null);
    }

    /**
     * 失败返回结果，可以自定义提示信息
     * @param message 提示信息
     */
    public static <T> CommonResult<T> failed(String message) {
        return new CommonResult<>(ResultCode.FAILED.getCode(), message, null);
    }

    /**
     * 参数验证失败返回结果
     */
    public static <T> CommonResult<T> validateFailed() {
        return new CommonResult<>(ResultCode.VALIDATE_FAILED.getCode(), ResultCode.VALIDATE_FAILED.getMessage(), null);
    }

    /**
     * 参数验证失败返回结果，可以自定义提示信息
     * @param message 提示信息
     */
    public static <T> CommonResult<T> validateFailed(String message) {
        return new CommonResult<>(ResultCode.VALIDATE_FAILED.getCode(), message, null);
    }

    /**
     * 未登录返回结果
     * @param data 返回的结果
     */
    public static <T> CommonResult<T> unauthorized(T data) {
        return new CommonResult<>(ResultCode.UNAUTHORIZED.getCode(), ResultCode.UNAUTHORIZED.getMessage(), data);
    }

    /**
     * 未授权返回结果
     * @param data 返回的结果
     */
    public static <T> CommonResult<T> forbidden(T data) {
        return new CommonResult<>(ResultCode.FORBIDDEN.getCode(), ResultCode.FORBIDDEN.getMessage(), data);
    }

    public long getCode() {
        return code;
    }

    public void setCode(long code) {
        this.code = code;
    }

    public String getMessage() {
        return message;
    }

    public void setMessage(String message) {
        this.message = message;
    }

    public T getData() {
        return data;
    }

    public void setData(T data) {
        this.data = data;
    }
}
```

　　3、使用通用返回结果类封装响应

```java
@RequestMapping("/selectStudents")
public CommonResult<List> selectStudents(){
    List<Student> list = studentService.selectStudents(1, 20);
    return CommonResult.success(list,"获取学生信息成功");
}
```

## 二、返回异常封装

　　为了让异常返回到前端时也有一个统一的格式，我们需要对异常响应进行封装。

　　1、创建自定义异常

```java
package com.zyg.exception;

import com.zyg.result.IErrorCode;

/**
 * @Author: zyg
 * @Date: 2023/4/26 17:43
 * @Version: v1.0
 * @Description: 弱密码异常
 */
public class WeekPassWordException extends RuntimeException {

    public WeekPassWordException(String message) {
        super(message);
    }

    public WeekPassWordException(String message, Throwable cause) {
        super(message, cause);
    }

    public WeekPassWordException(Throwable cause) {
        super(cause);
    }
}
```

　　2、创建全局异常处理类

```java
package com.zyg.handler;

import com.zyg.exception.WeekPassWordException;
import com.zyg.result.CommonResult;
import org.springframework.web.bind.annotation.ControllerAdvice;
import org.springframework.web.bind.annotation.ExceptionHandler;
import org.springframework.web.bind.annotation.ResponseBody;

/**
 * @Author: zyg
 * @Date: 2023/4/26 17:53
 * @Version: v1.0
 * @Description: 全局异常处理类
 */
@ControllerAdvice
public class GlobalExceptionHandler {

    @ResponseBody
    @ExceptionHandler(WeekPassWordException.class)
    public CommonResult handleWeekPasswordException(WeekPassWordException e) {
        return CommonResult.validateFailed(e.getMessage());
    }
}
```

　　3、抛出自定义异常，触发全局异常处理方法，返回CommonResult封装的异常信息

```java
@RequestMapping("/validatePwd")
public CommonResult<List> validatePwd(String pwd){
    try {
        if(pwd.length() < 8) {
            throw new WeekPassWordException("密码长度不能小于8位");
        }
    }catch (WeekPassWordException e) {
        throw new WeekPassWordException("密码长度不能小于8位");
    }

    return CommonResult.success(null,"密码验证成功");
}
```
