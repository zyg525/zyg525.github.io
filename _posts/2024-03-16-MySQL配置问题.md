## 一、修改数据库密码

### 修改root用户密码

　　root用户一般有两个权限，分别是`root@localhost`和`root@%`，前者针对本地客户端，后者针对远程客户端。修改root用户密码时，需要在客户端执行以下命令：

```sql
set password for root@localhost = password('123456');
set password for root@'%' = password('123456');
```
